## なぜ作るか

* 公式で日本語訳ページを作る企業あんまないでしょ
* 原文を自分で読んでも翻訳サイト使っても機微が分からん
  * 翻訳先言語の多数の目があれば、とは思いつつ誰が訳すねんっていうね
